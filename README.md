# Student Records Management System

# Project Description:

The Student Records Management System is a web-based application developed using PHP, JavaScript, and CSS that allows users to efficiently manage student information in an organized and user-friendly manner. This project is ideal for schools, colleges, or educational institutes that want a simple system to keep track of their students.

# Key Features:

Add, Edit, and Delete Students:
Easily manage student records including Name, Email, Phone Number, Age, Class, and Last Updated timestamp.

Search Functionality:
Quickly find students by name using the built-in search bar.

Responsive Dark Mode UI:
Modern and clean interface with a dark theme for better readability and aesthetics.

Real-Time Updates:
All operations are performed dynamically with instant updates using JavaScript and AJAX.

Data Validation:
Input validation for email, phone number, and age ensures accurate and consistent data entry.

Notifications:
Informative toast notifications provide instant feedback on adding, updating, or deleting records.

# Technologies Used:

Frontend: HTML, CSS (Dark Mode), JavaScript

Backend: PHP (Session-based storage)

Fonts: Google Fonts - Inter & Outfit

# Advantages:

Lightweight and easy to set up with Laragon or XAMPP.

No database setup required; session-based storage simplifies testing.

Extensible to include more fields, export options, or integration with databases like MySQL.

Enhances organization and accessibility of student information.

# Potential Enhancements:

Export student data to CSV or PDF

Sort students by Name, Class, or Age

Highlight missing or invalid data

Integrate with a backend database for permanent storage

# How to Run:

Install Laragon or XAMPP on your system.

Place the project folder in the www (Laragon) or htdocs (XAMPP) directory.

Open index.php in a browser via localhost.

Start adding, editing, or searching student records immediately.
