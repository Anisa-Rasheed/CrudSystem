# Student Records Management System 

The Student Records Management System is a simple web-based application built using PHP, JavaScript, and CSS that allows users to efficiently manage student information. This simplified version focuses on the essential details of students: Name, Age, and Class, making it easy to track and organize basic student records.

This project is ideal for small educational institutes, schools, or individual teachers who need a lightweight system to manage students without a database setup.

# Key Features:

Add, Edit, and Delete Students:
Quickly manage student records including Name, Age, and Class.

Search Functionality:
Easily search students by name using the integrated search bar.

Responsive Dark Mode UI:
Modern, clean interface with a dark theme for better readability.

Real-Time Updates:
Changes are immediately reflected in the table without page reloads.

Notifications:
Instant toast notifications indicate successful addition, update, or deletion of records.

# Technologies Used:

Frontend: HTML, CSS (Dark Mode), JavaScript

Backend: PHP (Session-based storage)

# Advantages:

Simple and lightweight, easy to set up using Laragon or XAMPP

No database required; uses session storage for easy testing

Focused on essential student information for quick management

Extensible for additional fields if needed

# How to Run:

Install Laragon or XAMPP on your system.

Place the project folder in the www (Laragon) or htdocs (XAMPP) directory.

Open index.php in a browser via localhost.

Start adding, editing, or searching student records immediately.
